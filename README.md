### 1. Introducción

En el desarrollo del presente taller se aplicaron herramientas TIC con el objetivo de fortalecer habilidades en el uso de software ofimático y plataformas colaborativas. Estas herramientas permiten mejorar la productividad, la organización de la información y el trabajo en equipo.

---

### 2. Objetivo

Aplicar funcionalidades de herramientas ofimáticas y servicios TIC, siguiendo buenas prácticas y procedimientos establecidos.

### 3. Desarrollo de la actividad

#### 3.1 Procesador de texto

Se utilizó Microsoft Word / Google Docs para:

* Elaborar documentos formales
* Aplicar formato (títulos, tablas, imágenes)
* Organizar la información de manera clara

#### 3.2 Hoja de cálculo

Se trabajó con Microsoft Excel / Google Sheets para:

* Registrar datos
* Aplicar fórmulas básicas
* Generar gráficos para análisis de información

#### 3.3 Software de diagramación

Se empleó Lucidchart / Microsoft Visio para:

* Crear diagramas de flujo
* Representar procesos de manera visual
* Facilitar la comprensión de la información

#### 3.4 Herramientas colaborativas

Se utilizó GitHub como plataforma principal para:

* Compartir archivos y documentos
* Controlar versiones de los trabajos realizados
* Colaborar en equipo de forma organizada

Además, se apoyó el trabajo con herramientas como:

* Google Drive para almacenamiento
* Microsoft Teams para comunicación

### 5. Conclusiones

El uso de herramientas TIC facilita la gestión de la información y el trabajo en equipo. Plataformas como GitHub permiten llevar un control ordenado de los proyectos, mientras que las herramientas ofimáticas permiten presentar la información de forma clara y profesional.

### 6. Recomendaciones
